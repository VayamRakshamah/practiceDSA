Problems to solve (in order):

1. **Two Sum** (Leetcode 1)
   Difficulty: Easy
   Pattern: Hash Map
   Approach:
   - Use HashMap to store value → index
   - For each number, check if (target - number) exists in map
   - If yes, return both indices
   Time: O(n), Space: O(n)

2. **Duplicate Removal (Remove Duplicates from Sorted Array)** (Leetcode 26)
   Difficulty: Easy
   Pattern: Two Pointers
   Approach:
   - Left pointer for new array position
   - Right pointer for scanning
   - When number changes, increment left and set value
   Time: O(n), Space: O(1)

3. **Contains Duplicate** (Leetcode 217)
   Difficulty: Easy
   Pattern: Hash Set
   Approach:
   - Iterate through array
   - Add to set if not present
   - If already in set, return True
   - If finish iteration, return False
   Time: O(n), Space: O(n)

4. **Best Time to Buy and Sell Stock** (Leetcode 121)
   Difficulty: Easy
   Pattern: Single Pass with Min Tracking
   Approach:
   - Track minimum price seen so far
   - At each price, calculate profit if sold here
   - Keep track of max profit
   Time: O(n), Space: O(1)

5. **Valid Palindrome** (Leetcode 125)
   Difficulty: Easy
   Pattern: Two Pointers with String Manipulation
   Approach:
   - Clean string (alphanumeric only, lowercase)
   - Two pointers from ends
   - Compare characters, move inward
   - If mismatch, return False
   Time: O(n), Space: O(n) for cleaned string
